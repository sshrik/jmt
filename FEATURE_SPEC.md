# JMT Cursor - 투자 전략 백테스팅 플랫폼

## 🎯 프로젝트 개요

복잡한 투자 전략을 시각적으로 설계하고 백테스팅할 수 있는 웹 플랫폼입니다.

---

## ✅ 구현 완료된 기능

### 🔄 **버전 관리 및 자동 백테스트**

- **버전 생성 후 백테스트 자동 실행**: 새 버전 생성 시 토글을 통해 자동으로 백테스트 실행
- **버전별 백테스트 이력 관리**: 동일 버전에 대해 여러 백테스트 결과 저장 및 조회
- **버전 저장 시스템**: 새 버전 생성 시 올바른 프로젝트 저장 및 영속화

### 🎨 **UI/UX 개선**

- **다크모드 최적화**:
  - 투자 수익률 색상을 다크모드에 맞게 조정 (양수/음수 구분)
  - 주식 정보 카드 색상 개선 (Stock Trend 페이지)
  - CSS 클래스 기반 테마 대응 시스템
- **대시보드 UI 개선**:
  - 순위 칩(#1, #2...)과 메뉴 버튼 정렬 개선
  - 프로젝트 카드 레이아웃 최적화
- **날짜 선택기 개선**:
  - 백테스트 설정에서 달력이 항상 아래로 열리도록 수정
  - 주의 시작을 일요일로 설정

### 🚀 **성능 최적화**

- **프로젝트 수정 폼 성능 개선**:
  - 프로젝트 이름/설명 폼을 별도 컴포넌트(`ProjectInfoForm`)로 분리
  - 실시간 저장을 제거하고 버전 생성 시에만 저장
  - 디바운싱 및 메모이제이션 적용

### 📊 **백테스트 기능 강화**

- **백테스트 상세 정보 모달**:
  - 백테스트 이력 카드 클릭 시 상세 정보 표시
  - 테스트 파라미터, 성과 지표, 거래 내역 포함
  - "동일 조건으로 다시 테스트" 기능
- **거래 내역 개선**:
  - NaN 값 오류 해결 (금액, 수수료 정상 표시)
  - 거래 사유(`reason`) 필드 추가로 투명성 향상
- **대시보드 수익률 표시**: 가장 최근 백테스트 결과로 수익률 계산 및 표시

### 🛠 **시스템 안정성**

- **데이터 호환성 및 복구**:
  - localStorage 데이터 호환성 보장 (기존 단일 객체 → 배열 변환)
  - 프로젝트 데이터 자동 복구 시스템
  - 고속도로 매매법 예시 프로젝트 제공
- **빌드 시스템 개선**:
  - TypeScript 컴파일 에러 완전 해결
  - ESLint 에러 0개 달성
  - deprecated `.substr()` → `.substring()` 교체
  - `any` 타입을 `unknown`으로 교체하여 타입 안전성 향상
- **테스트 안정성**: 비결정적 테스트 오류 해결 (타임스탬프 기반 정렬 문제)

### 📈 **고속도로 매매법 구현**

- **실전 투자 전략 예시**:
  - 0-2% 하락/상승 시 기본 매매
  - 2-4% 하락/상승 시 추가 매매
  - 4%+ 하락/상승 시 대량 매매
- **전략 블록 활성화**: 모든 조건/액션 블록에 `enabled: true` 설정
- **백테스트 엔진 개선**: 디버그 로그 추가로 거래 추적 가능

### 📖 **사용자 매뉴얼 시스템**

- **통합 도움말 페이지**: 사이드바 "도움말" 섹션에 사용자 매뉴얼 추가
- **종합 가이드 제공**:
  - 5분 만에 시작하기 (빠른 시작 가이드)
  - 전략 설계 방법 (룰 기반 vs 플로우차트 비교)
  - 백테스트 실행 및 결과 분석 가이드
  - 주요 기능 활용법 (대시보드, 주식 추이, 환경설정, 버전 관리)
  - 팁과 모범 사례 (성공적인 전략 설계 방법론)
  - 자주 묻는 질문 (FAQ)
- **사용자 친화적 UI**:
  - Accordion 기반 카테고리별 정리
  - 카드 레이아웃과 아이콘으로 시각적 구분
  - 다크모드 완전 지원
  - GitHub Issues 및 개발자 프로필 직접 연결

## 🔧 기술 스택

### Frontend

- **Framework**: React 18 + TypeScript
- **Router**: TanStack Router v1
- **UI Library**: Mantine v7
- **State Management**: Zustand (ProjectStore, VersionStore)
- **Build Tool**: Vite
- **Styling**: CSS Modules + Mantine Theme

### 데이터 저장

- **Primary**: localStorage (브라우저 로컬 저장)
- **Backup**: Mock 데이터 자동 생성 시스템

### 테스팅

- **Test Runner**: Node.js + TSX
- **Test Files**: `tests/` 디렉토리
